# 💼 Portfólio 

Este é meu projeto de portfólio desenvolvido com React + Material UI (MUI), criado para apresentar alguns dos projetos que venho desenvolvendo na área de desenvolvimento web.

O objetivo deste projeto é reunir minhas principais aplicações, demonstrando minhas habilidades com interfaces modernas, componentização, responsividade e boas práticas no front-end.


## 🚀 Tecnologias utilizadas

- React
- Material UI (MUI)
- JavaScript
- HTML5
- CSS3


## 📂 Projetos apresentados

## 📚 SGL 

Projeto desenvolvido com o objetivo de organizar e acompanhar leituras de forma prática e visual, permitindo o gerenciamento completo dos livros lidos, em leitura e a serem lidos. A aplicação oferece uma visão clara do progresso de leitura, facilitando o controle pessoal e incentivando o hábito da leitura. 

### ✨ Funcionalidades 

- Cadastro de livros 
- Edição e exclusão de leituras 
- Organização por status 
- Exibição das capas dos livros 
- Interface intuitiva e responsiva 

### 🛠️ Conceitos aplicados 

- CRUD 
- Componentização 
- Organização de dados 
- Estruturação de interfaces 
- Boas práticas de código 

## 🛍️ Brechó 

Projeto desenvolvido para apresentar os bens disponíveis para os servidores da Prefeitura Municipal de Vitória da Conquista (PMVC), permitindo visualização e solicitação dos itens de forma organizada e intuitiva. A aplicação busca facilitar o acesso às informações, promovendo transparência e melhor aproveitamento dos recursos. 

### ✨ Funcionalidades 

- Listagem de bens 
- Visualização de detalhes 
- Fluxo de solicitação 
- Interface moderna e responsiva 

### 🛠️ Conceitos aplicados 

- Componentização 
- Organização de layouts 
- Consumo de dados 
- Responsividade 
- Estruturação front-end 

## 🎨 Layout e Design

O projeto foi desenvolvido utilizando Material UI para criação de uma interface moderna, limpa e responsiva, focando na experiência do usuário e organização visual.
